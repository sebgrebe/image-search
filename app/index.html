Image Search Abstraction Layer

This app lets you search for images on the Google Custom Search API and returns the images' link, a description of them, and the URL of their webpage.

Just type in /search/[SEARCH TERM] at the end of this page's URL. You can use the 'offset' parameter to determine which set of results is displayed

Example

Input: 'https://image-search.herokuapp.com/search/cats?offset=3'

Output: A list of the images displayed on page 3 of the search results for 'cats'.
